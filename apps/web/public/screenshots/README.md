# PWA Screenshots

This directory contains screenshots for the Progressive Web App (PWA) installation prompts.

## Required Screenshots

### Desktop (Wide Form Factor)
- **Filename**: `desktop-dashboard.png`
- **Size**: 1280x720 pixels
- **Purpose**: Shown in desktop PWA installation prompts
- **Content**: Dashboard view showing key HR metrics

### Mobile (Narrow Form Factor)
- **Filename**: `mobile-dashboard.png`
- **Size**: 750x1334 pixels
- **Purpose**: Shown in mobile PWA installation prompts
- **Content**: Mobile-optimized dashboard view

## Screenshot Guidelines

- **Format**: PNG for best quality
- **Content**: Should showcase the app's main features
- **Clarity**: High resolution, no pixelation
- **Branding**: Should match the app's visual identity
- **Real data**: Use realistic sample data (avoid Lorem Ipsum)

### How to Capture Screenshots

1. Run the app in development mode: `bun run dev:web`
2. Open in browser at full desktop size (1280x720)
3. Navigate to the dashboard
4. Use browser developer tools to capture screenshot (Cmd/Ctrl + Shift + P → "Screenshot")
5. For mobile, use device emulation (iPhone 11 Pro Max or similar)
6. Save files with the names listed above

### Alternative Tools

- [Screely](https://www.screely.com/) - Add browser mockups
- [Rotato](https://rotato.app/) - 3D device mockups
- [Shotsnapp](https://shotsnapp.com/) - Device frame mockups
